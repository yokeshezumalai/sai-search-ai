The Problem

People can't understand
their own spending

Financial data exists. Insight doesn't. People generate thousands of transactions per year but have no natural way to interrogate that history.

  • Legacy banking apps offer static, uncustomisable category views
  • Spreadsheet exports require manual filtering and pivot tables
  • Keyword search misses context — "fuel" won't find "Shell Top-Up"
  • No product understands temporal language: "last 3 months of dining"
  • Enterprise analytics tools are priced out of consumer and SMB reach

The opportunity we're seizing

Large language models made natural language interfaces mainstream. Vector databases made semantic search fast and cheap. SAI combines both into a single, embeddable API that any fintech, neobank, or wealth app can integrate within a day.

Architecture

Five steps from
query to insight

I

Query Received

Natural language arrives at /api/search with an API key in the header.

II

Parse Intent

Regex and keyword rules extract the date range and spending category from the query.

III

Vector Encoding

The query is encoded into a 384-dim float16 vector by the MiniLM model.

IV

Hybrid Search

Milvus combines cosine similarity with scalar date and category filters — top 100 hits returned.

V

Aggregate & Return

Results grouped by merchant, totals computed, highest spend surfaced, response serialised.

Security

Isolation by design

Per-user data architecture

Every API key deterministically maps to an isolated Milvus collection. No cross-contamination between users is architecturally possible — not just a policy, a structural guarantee.

The collection name is derived as transactions_{api_key[:16]} — lightweight, collision-resistant, never storing the full key server-side.

  • Obtain your API key and include it as X-API-Key
  • Upload your CSV via POST /api/ingest
  • Your Milvus collection is auto-created and indexed
  • All queries are scoped exclusively to your collection
  • Monitor status at GET /api/ingest/status
// Include in every authenticated request
