# SAI — Search AI

A beautiful, elegant AI-powered search interface built with Claude Code.

![SAI Search AI](https://img.shields.io/badge/Built%20with-Claude%20Code-blue?style=flat-square)

## Overview

SAI (Search AI) is a sophisticated search interface that combines traditional search functionality with AI-powered capabilities. The design features an elegant, minimalist aesthetic with a focus on typography and user experience.

## Features

- AI-powered search functionality
- Elegant serif typography (Cormorant Garamond, Tenor Sans)
- Smooth animations and transitions
- Responsive design
- Dark code blocks with syntax highlighting
- Keyboard shortcuts support
- Search history and suggestions

## Tech Stack

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Design**: Custom CSS with CSS variables
- **Built with**: Claude Code (Anthropic)

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/sai-search-ai.git
   ```

2. Open `index.html` in your browser

3. Start searching!

## Keyboard Shortcuts

- `Cmd/Ctrl + K` - Focus search
- `Esc` - Clear search / Close modals

## Design System

### Colors
- Primary: `#0d0d0d` (Ink Black)
- Background: `#faf8f4` (Paper White)
- Accent: `#b08a3e` (Gold)
- Code Background: `#111009`

### Typography
- Headings: Cormorant Garamond
- Body: Tenor Sans
- Code: DM Mono
